# 🎉 장현제님 생일 축하 웹사이트

장현제님의 생일을 축하하는 아름답고 우아한 웹사이트입니다.

## 설치 및 실행

```bash
# 패키지 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

## 기술 스택

- React 18
- TypeScript
- Vite
- CSS3 (애니메이션 및 그라데이션)

## 특징

- 🌈 아름다운 그라데이션 배경
- ✨ 컨페티 애니메이션
- 🎈 풍선 플로팅 효과
- 🎂 생일 케이크 애니메이션
- 💖 하트 비트 효과
- 📱 반응형 디자인
